Rodar no terminal npm i typescript

Iniciar projeto JS - npm init 

Iniciar aplicação Typescript npx tsc --init (cria arquivo de configuração)

ir no arquivo tsconfig e descomentar e configurar o atributo outdir (especificar a pasta que o arquivo JavaScript vai receber o codigo ts transpilado)

npx tsc

ir no Package json em Scripts e criar um atributo como "start": "npx tsc --watch" (sempre que salvar o código ele vai transpilar TS o código pra um arquivo JS)

Documentação: https://www.typescriptlang.org/download
